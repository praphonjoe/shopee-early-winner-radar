# ก่อนกระแส · Shopee → Radar (Chrome Extension)

ปุ่มลอยบนหน้าสินค้า Shopee — คลิกเดียวดูดข้อมูลจริง (ยอดขาย/รีวิว/ราคา) เข้าแอป
ทำงาน **ในเบราว์เซอร์คุณเอง** จึงไม่โดน Shopee บล็อก (เพราะคุณคือผู้ใช้จริง)

## วิธีติดตั้ง (ครั้งเดียว ~1 นาที)
1. เปิด Chrome พิมพ์ที่ช่อง URL: **`chrome://extensions`**
2. เปิดสวิตช์ **Developer mode** (มุมขวาบน)
3. กด **Load unpacked** → เลือกโฟลเดอร์ **`extension`** นี้
4. เสร็จ! จะเห็นส่วนขยาย "ก่อนกระแส · Shopee → Radar"

## วิธีใช้
1. เปิดหน้าสินค้าใน **shopee.co.th** (หน้าที่มี `-i.เลข.เลข` ใน URL)
2. รอหน้าโหลดสักครู่ → มุมขวาล่างจะมีปุ่ม **`＋ เข้าเรดาร์`** (เขียว = พร้อม)
3. กดปุ่ม → ข้อมูลสินค้าจะถูกส่งเข้าแอปทันที (เด้ง ✅)
4. เปิดแอป **shopee-early-winner-radar.vercel.app** → สินค้าจะอยู่ในอันดับแล้ว

## หมายเหตุ
- ปุ่มจะขึ้นเฉพาะ "หน้าสินค้า" เท่านั้น (ไม่ขึ้นหน้าค้นหา/หน้าร้าน)
- ถ้าปุ่มค้างที่ "📡 กำลังอ่านหน้า…" ให้เลื่อนหน้าลงนิดนึงหรือรีเฟรช เพื่อให้ Shopee โหลดข้อมูลครบ
- ข้อมูลที่ Shopee ให้: ชื่อ · ราคา · ยอดขายสะสม · รีวิว · เรทติ้ง
  (ไม่มี "ครีเอเตอร์/Live" เพราะเป็นสัญญาณของ TikTok — สูตรคะแนนฝั่ง Shopee ปรับแยกต่างหาก)
