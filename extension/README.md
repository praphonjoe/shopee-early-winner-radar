# ก่อนกระแส · Shopee → Radar (Chrome Extension) v2

**เก็บสินค้า Shopee ยกล็อต** จากหน้าค้นหา/หมวด/มาแรง เข้าแอป แล้วให้แอป **จัดอันดับ "วันนี้ทำอะไรดี" ให้เอง**
ทำงาน **ในเบราว์เซอร์คุณเอง** จึงไม่โดน Shopee บล็อก (คุณคือผู้ใช้จริง)

## วิธีติดตั้ง (ครั้งเดียว ~1 นาที)
1. เปิด Chrome → พิมพ์ **`chrome://extensions`**
2. เปิด **Developer mode** (มุมขวาบน)
3. **Load unpacked** → เลือกโฟลเดอร์ **`extension`** นี้
4. *(ถ้าเคยติดตั้ง v1 แล้ว กด **Reload** ที่การ์ด extension เพื่ออัปเป็น v2)*

## วิธีใช้ — หา "สินค้าก่อนกระแส" แบบไม่ต้องเปิดทีละตัว
1. ไป **shopee.co.th** → **เสิร์ชคำที่สนใจ** (เช่น "เครื่องนวด", "ของแต่งบ้าน") หรือเปิด **หน้าหมวด / Flash Sale / มาแรง**
2. มุมขวาล่างจะมีแผง **"ก่อนกระแส · เรดาร์"** โชว์ตัวเลขสินค้าที่เก็บได้
3. กด **🔄 เลื่อนเก็บอัตโนมัติ** → มันจะเลื่อนหน้าเก็บสินค้าให้เอง (หลายสิบ–ร้อยตัว)
4. กด **⬇️ ส่งเข้าเรดาร์ (N)** → ส่งทั้งหมดเข้าแอปทีเดียว
5. เปิดแอป **shopee-early-winner-radar.vercel.app** → **แอปจัดอันดับให้แล้ว** ✅

## หมายเหตุ
- เก็บจากหน้าไหนก็ได้ที่มีลิสต์สินค้า (ค้นหา/หมวด/ร้าน/มาแรง/สินค้าใกล้เคียง)
- กันข้อมูลซ้ำอัตโนมัติด้วย `src_id` (เก็บซ้ำ = อัปเดตตัวเดิม)
- Shopee ให้: ชื่อ · ราคา · ยอดขายสะสม · รีวิว · เรทติ้ง (ไม่มีครีเอเตอร์/Live ซึ่งเป็นของ TikTok)
  → สูตรคะแนนฝั่ง Shopee จะปรับให้ใช้เฉพาะสัญญาณเหล่านี้
