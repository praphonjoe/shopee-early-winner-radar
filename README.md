# ก่อนกระแส · Early Winner Radar 📡

แอปมือถือหาสินค้า **“ก่อนกระแส”** (early-stage winner) สำหรับ Shopee / TikTok Shop —
ถามคำเดียวว่า *“วันนี้ทำสินค้าอะไรดี?”* แล้วระบบจัดอันดับ + ให้คะแนน + แนะนำมุมคอนเทนต์ให้ตัดสินใจได้ทันที

## สถาปัตยกรรม
- **Frontend** — HTML/CSS/JS ล้วน (ไม่ต้อง build) เปิดบนมือถือได้ เพิ่มลงหน้าจอโฮมเป็นแอปได้
- **Supabase** — เก็บรายการสินค้า (ตาราง `products`) เพิ่มแล้วบันทึกถาวร ใช้ได้ทุกเครื่อง
- **Vercel** — deploy อัตโนมัติทุกครั้งที่ push ขึ้น GitHub

## ระบบให้คะแนน (เต็ม 100)
| มิติ | น้ำหนัก |
|------|--------|
| โมเมนตัมยอดขาย (กำลังโต) | 25 |
| ครีเอเตอร์ยังน้อย | 20 |
| ทำคอนเทนต์ปังง่าย (ไม่แบน) | 20 |
| มาร์จิ้น & ช่วงราคา | 15 |
| หลักฐานดีมานด์ (รีวิว/เรทติ้ง) | 10 |
| คู่แข่งไม่แน่น (จำนวนร้าน) | 10 |

**การแปลผล:** 80–100 ลุยเลย · 65–79 น่าสนใจ · 50–64 กลาง ๆ · <50 ผ่าน
(ปรับสูตร/น้ำหนักได้ในฟังก์ชัน `score()` ที่ `app.js`)

## ตั้งค่า
1. สร้างโปรเจกต์ที่ [supabase.com](https://supabase.com) → SQL Editor → วางไฟล์ [`supabase/schema.sql`](supabase/schema.sql) แล้ว Run
2. เอา **Project URL** + **anon key** (Project Settings → API) ใส่ใน [`config.js`](config.js)
3. push ขึ้น GitHub → Vercel deploy อัตโนมัติ

> `anon key` เปิดเผยได้ ปลอดภัยด้วย Row Level Security — ไม่ใช่ service_role key

## รันในเครื่อง
```bash
npx serve .
# หรือ
python3 -m http.server 8000
```
